# rougerRoll

一个掷骰子决定攻防伤害的 Boss 战 Web 小游戏。

## 运行方式

直接在浏览器中打开 `index.html`，或在项目目录执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 玩法

- 玩家初始生命值为 20，Boss 初始生命值为 20。
- 点击“玩家行动”后会进入一回合：
  - 玩家先掷骰子，点数即对 Boss 造成的伤害。
  - 若 Boss 未被击败，Boss 会掷骰子反击，点数即对玩家造成的伤害。
- 每次 Boss 被击败后会立即重生并强化：
  - Boss 攻击骰上限 +1（D10 → D11 → D12...）。
  - Boss 最大生命值 +10，且会以满血重生。
- 玩家生命值降到 0 时战斗结束。
- 点击“重新开始”可重置战斗。
