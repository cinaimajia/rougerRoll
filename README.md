# rougerRoll

一个掷骰子决定攻防伤害的 Boss 战 Web 小游戏。

## 运行方式

直接在浏览器中打开 `index.html`，或在项目目录执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 玩法

- 玩家和 Boss 初始生命值都为 20。
- 点击“发起攻击”后会进入一回合：
  - 玩家先掷骰子，点数即对 Boss 造成的伤害。
  - 若 Boss 未被击败，Boss 会掷骰子反击，点数即对玩家造成的伤害。
- 任何一方生命值降到 0，战斗结束：
  - Boss 归零：玩家胜利。
  - 玩家归零：游戏失败。
- 点击“重新开始”可重置战斗。
